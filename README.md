# Ragnarok Online 2.5D Clone (MVP)

## Setup instructions

1. **Backend**:
   - `npm install`
   - Create a `.env` file at the root with `SUPABASE_URL` and `SUPABASE_KEY`.
   - Execute `supabase-init.sql` in your Supabase SQL editor.
   - Run `npm run dev` to start the server.

2. **Frontend**:
   - The game runs at `http://localhost:3000`.
